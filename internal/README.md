# Go Internals

This folder contains the Go implementation for the relay, control plane, CLI support, storage, and protocol layers.

Subfolders are organized by subsystem:

- `auth/`
- `cli/`
- `client/`
- `config/`
- `controlplane/`
- `db/`
- `models/`
- `protocol/`
- `proxy/`
- `server/`
- `utils/`
